React Date Picker - Bootstrap 4 & Material Design

React Bootstrap date picker is a plugin that adds the function of selecting time without the necessity of using custom JavaScript code.

This documentation may contain syntax introduced in the MDB React 5.0.0 and can be incompatible with previous versions. For old timepicker documentation please follow the link.

Themes Defaults

You can set a themes to DatePicker Default with prop theme.

Available colors:

  • warning
  • danger
  • success
  • primary
  • dark
  • info
  • secondary
