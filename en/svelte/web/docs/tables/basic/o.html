Svelte Bootstrap Table

Svelte Table - Bootstrap 4 & Material Design

Svelte tables allow you to aggregate a huge amount of data and present it in a clear and orderly way.

Bootstrap Svelte tables provide additional benefits like responsiveness and the possibility of manipulating the table styles.
